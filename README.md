# Portfolio of Atul Raj Gupta

A fully responsive personal portfolio website for Atul Raj Gupta, a Senior React Native Developer. Built with modern web standards to showcase professional experience, technical skills, and key projects.

## Overview
This project is a clean, minimal, and highly professional vCard/Portfolio website. It features a sticky sidebar, an interactive navigation menu, and a filtered portfolio gallery.

## Features
- **Profile Sidebar**: Quick access to contact information and professional social links.
- **About Section**: Detailed professional summary and service offerings.
- **Resume**: Chronological experience timeline and core technical skills visualization.
- **Project Portfolio**: Filterable project gallery showcasing React Native and mobile app development work.
- **Contact Form**: Integrated contact form and Google Maps location view.
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.

## Installing vCard

To install **vCard**, follow these steps:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/vcard-personal-portfolio.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/vcard-personal-portfolio.git
```
